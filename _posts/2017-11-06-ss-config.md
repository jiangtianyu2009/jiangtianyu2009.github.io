Restart SS server command:
```
ssserver -h
ssserver -c /etc/shadowsocks.json -d restart
```

Content of `/etc/shadowsocks.json`:
```
{
    "server":"x.x.x.x",
    "server_port":8000,
    "local_port":1080,
    "password":"******",
    "method":"aes-256-cfb"
}
```
